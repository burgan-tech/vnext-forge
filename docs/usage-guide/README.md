# vNext Forge Studio — Usage Guide

vNext Forge Studio is a workflow design and management tool delivered as a VS Code extension. It provides a visual workflow designer, component editors (tasks, schemas, views, functions, extensions), an integrated C# script editor, Quick Run runtime testing, and project scaffolding for the vNext engine ecosystem.

## Quick Start

1. Install the **vNext Forge Studio** extension from the VS Code Marketplace (or from a `.vsix` file).
2. Open a folder containing a `vnext.config.json` file — the extension activates automatically.
3. Right-click any component `.json` file in the Explorer and select **Forge: Open with vNext Forge**, or use the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) and type **Forge**.
4. Start designing workflows, editing components, and testing against a live runtime with Quick Run.

## Desktop App

vNext Forge Studio is also available as a standalone desktop application. The desktop app uses the same React-based UI and designers as the VS Code extension — everything documented in this guide (workflow canvas, component editors, Quick Run, CSX editor) works identically in both shells. Desktop-specific features and screenshots will be provided separately.
